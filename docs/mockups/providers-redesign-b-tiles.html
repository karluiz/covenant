Why this works

  • Scannable overview. Every provider gets a tile with logo, name, endpoint, and a status pill — you spot the failing one (red 401) without opening anything.
  • "Default" treated as first-class. The default provider gets an accent border + badge. No more hunting for which one is actually in use.
  • Edit happens in a focused drawer. Form work isn't competing with the list. Drawer animates in, closes with ✕ or Esc — feels like Linear / Vercel project settings.
  • Status is meaningful. "Connected", "Not tested", "401 Unauthorized · rotate key" — the panel is also a health dashboard, not just an editor.
  • Filter built in. When you grow to 15+ providers (one per model family, prod/staging, etc.) the filter saves you. Today's stack has no answer for that scale.
  • Tradeoff: requires drawer infrastructure (overlay, escape handling, focus trap). More code than Direction A. Tile grid also needs ≥ 720 px to keep 3 columns — narrower windows drop to 2 or 1.

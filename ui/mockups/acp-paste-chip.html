Rules

Threshold
≥ 600 characters or ≥ 8 lines. Anything shorter still pastes inline — a two-line snippet doesn't deserve ceremony.
Token
[Pasted text #1 +42 lines], inserted at the caret. It is ordinary text: you can move it, wrap prose around it, or delete it.
Deleting the token
Drops the paste on send. Same contract as an @mention — the text is the source of truth, not a hidden list.
The chip
Sits in the strip images already use, above the composer. Clicking it removes both the chip and its token.
On send
Each surviving token is replaced by its exact content before the prompt reaches the agent. The agent sees the full text; you never did.
